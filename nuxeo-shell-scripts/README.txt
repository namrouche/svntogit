About
-----

This HG repository contains some scripts that can be run from the Nuxeo Shell.

How to use the scripts
----------------------

Each script should be documented in its header.

To use these scripts:

- download them
- call them from the Nuxeo Shell

from interactive mode use :
script --file <path_to_your scripts> <args>

For more information about the Nuxeo Shell:

http://doc.nuxeo.org/5.1/books/nuxeo-book/html/administration.html#nuxeo-shell
